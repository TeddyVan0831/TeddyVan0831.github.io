# 12.96 万亿 Token 碾压 3.03 万亿：中国 AI 大模型，连续五周教美国做人

根据 OpenRouter 2026 年 3 月 30 日至 4 月 5 日数据，全球 AI 模型总调用量达 27 万亿 Token，环比增长 18.9%。其中中国以 12.96 万亿 Token 位居榜首，占比 48%，是美国 3.03 万亿的 4.28 倍。

## 全球前六名全是中国模型

1. 阿里 Qwen3.6 Plus — 4.6 万亿 Token
2. 小米 MiMo-V2-Pro — 3.08 万亿 Token
3. 阿里 Qwen3.6 Plus Preview — 1.64 万亿 Token
4. 阶跃星辰 Step 3.5 Flash — 1.26 万亿 Token
5. MiniMax M2.7 — 1.19 万亿 Token
6. DeepSeek V3.2 — 1.19 万亿 Token

## 免费策略与出海扩张

中国模型的崛起离不开激进的免费策略。阿里、百度等企业通过开放 API 和免费调用额度快速占领开发者心智。

## 从跟跑到领跑

2023 年，中国 AI 还在追赶美国；2026 年，中国已经实现反超。这一转变的背后是密集的技术迭代、庞大的应用场景，以及国家政策的持续支持。

## 挑战与展望

尽管数据亮眼，中国 AI 仍面临三大挑战：商业化路径、原创性突破、国际竞争格局。
